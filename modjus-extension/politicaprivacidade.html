Política de Privacidade — Modjus Extension

Última atualização: 15 de abril de 2025

A extensão Modjus foi desenvolvida com foco na melhoria da experiência de uso do Sistema Eletrônico de Informações (SEI) em ambientes institucionais, permitindo a execução de modelos complexos de documentos de forma segura e controlada.

1. Coleta de Dados

A extensão não coleta, armazena, transmite ou compartilha dados pessoais dos usuários. Toda a funcionalidade ocorre localmente no navegador, e os dados processados são exclusivamente aqueles apresentados nas páginas do SEI, acessadas pelo próprio usuário.

2. Permissões Utilizadas

  • activeTab: permite à extensão interagir com a aba ativa (ex: para injetar scripts no SEI e ativar a funcionalidade dos modelos). Essa permissão é usada exclusivamente para páginas do SEI.
  • host_permissions: a extensão interage somente com os seguintes domínios:
    • https://sei.trf2.jus.br
    • https://sei-apresentacao.trf2.jus.br
    • https://sei-desenvolvimento.trf2.jus.br
    • http://localhost em ambiente de desenvolvimento

3. Armazenamento de Dados

A extensão não utiliza armazenamento local nem remoto. Nenhum dado é salvo ou transmitido para servidores externos.

4. Compartilhamento de Dados

A extensão não compartilha dados com terceiros.

5. Segurança

A extensão é executada exclusivamente no navegador do usuário, sem conexões externas além dos domínios previamente listados. Todo o código da extensão pode ser inspecionado livremente pelos usuários ao instalar via Chrome Web Store.

6. Alterações nesta Política

Caso a política de privacidade seja alterada, uma nova versão será disponibilizada nesta página com a data de atualização revisada.
